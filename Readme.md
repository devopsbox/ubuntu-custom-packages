## Scripts for Deb Packages


This repository contains scripts that compile software and package them to DEB-packages.

## Usage

    git clone https://github.com/devopsbox/ubuntu-custom-packages.git
    cd ubuntu-custom-packages


# Ubuntu 12.04

  - OpenResty (Nginx + Lua)
